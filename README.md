# React + TypeScript + Vite

Запуск:
1) npm install
2) npm run dev / click 'dev' script

Стек:
1) React
2) TypeScript
3) Vite
4) React-router-dom
5) MUI

Дополнительно:
1) Вначале запрашивается информация по всем странам только по требуемым филдам
2) При клике по флагу - запрашивается уже вся информация по стране
